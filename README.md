# CoastTech Corporate Website

A clean, responsive corporate website built using HTML, CSS, and vanilla JavaScript.

## Features
- Executive white theme
- Responsive design (mobile & desktop)
- Sticky navigation with active link highlighting
- Smooth scrolling and fade-in animations
- Functional contact form (frontend only)
- Back-to-top button
- Social media integration

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
This project was built to demonstrate frontend development skills during interviews and industrial attachment applications.
